 Multi-Container Web Application

## 📌 Project Overview
This project demonstrates a multi-container application using Docker and Docker Compose. The system consists of a Node.js web application and a MongoDB database running in separate containers.

--

## ⚙️ Technologies Used
- Docker
- Docker Compose
- Node.js (Express)
- MongoDB

---

## 🧩 Architecture
- Web Service → Node.js application
- Database Service → MongoDB
- Communication via Docker internal network

---

## 🚀 Features
- Add data through a web interface
- Store data in MongoDB
- Retrieve and display stored data
- Multi-container setup using docker-compose

---

## ▶️ How to Run

```bash
docker compose up --build
